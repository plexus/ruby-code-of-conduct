This list intends to provide information about the availability of a Code of Conduct at Ruby conferences, or other approaches taken in order to help provide a safe and comfortable experience to everyone at a conference.

    What is this list about?

    We believe every conference should have a Code of Conduct and information about the presence of a Code of Conduct will be valuable to conference attendees, speakers and organizers.

    We re-publish and add onto the data from the great project RubyConferences.org with their permission, and according to the CC Attribution-NonCommercial-ShareAlike 4.0 License.

    If you find a Ruby conference missing, please file an issue or submit a pull request to the RubyConferences.org repository. If you find information about a Code of Conduct missing or out of date, please file an issue or submit a pull request on our repository.
